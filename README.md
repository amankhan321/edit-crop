# edit-crop

Browser-based landscape → portrait video reframing tool. Crop, reframe and export vertical clips (9:16, 4:5, 1:1, 3:4, 4:3) for Reels / Shorts / TikTok / X. Runs 100% client-side — no upload, no server.

**Live:** https://amankhan321.github.io/edit-crop/

## Features
- Aspect-ratio presets with a draggable, ratio-locked crop box (rule-of-thirds guides)
- "Fit + blur" mode — keeps the whole frame, blurred fill behind (nothing cut off)
- Resolution cap (match source / 1080 / 720, never upscales) + bitrate quality tiers
- Audio preserved, exports MP4 (Chrome/Edge) with WebM fallback
